Di halaman ini saya akan menjelaskan tentang isi yang ada di website ini, nah jadi di website ini saya akan isikan dengan Gambar-gambar dan beberapa Tips and Trick tentang photoshop. Di Photoshop ada banyak fitur yang bisa kita gunakan yang nanti akan kita bahas di halaman Tips Photoshop.

Biasanya saya menggunaakn Photoshop untuk membuat logo, membuat skin, dan membuat gambar gambar seperti lukisan. Nah diatas ini adalah contoh dari salah satu bagian untuk membuat skin, kita bisa lihat kalau ada beberapa gambar yang di gabung gabung menjadi satu bagian. Ntar caranya saya akan bahas di Tips Photoshop.

Karena tidak tahu mau bahas apa lagi di halaman ini, maka saya akan memberi beberapa element di html yang saya ketahui sejauh ini. Sekalian bisa jadi note agar tidak lupa :)

1) Font= gunanya untuk mengubah mau bagaimana tulisan dibuat, font ini bisa diisi dengan:color,size,body,dll.
2) Span= mirip dengan font hanya saja biasanya aturnya di css bukan html.
3) h(1-6)= gunanya untuk membuat heading, yang paling besar mulai dari h1 sampai ke yang paling kecil h6.
4) hr= gunanya untuk membuat garis yang panjang, warnanya dan ukuran bisa diubah.
5) head= ini biasa untuk memasukan apa yang ingin diisi di html ini, dengan ini kita bisa isi css di html (style) dan title untuk judul halaman.
6) title= untuk menentukan judul halaman.
7) style= untuk memulai codingan class.
8) body= tempat untuk mengisi halaman yang akan kita buat.
9) p= untuk membuat paragraf
10) div= paling enak diingat div ini adalah grouping jadi kita bisa gabung beberapa element ke 1 grup :)
11) ul= untuk memulai listing (li)(ol)
12) li= untuk membuat list dengan simbol di kirinya
13) ol= untuk membuat list dengan angka di kirinya
14) center= untuk membuat element berada di tengah
15) html= awal mula yang harus kita ketik
16) !DOCTYPE html= paling awal dari yang kita ketik (opsional)
17) link= ini untuk memasukan file seperti css, biasanya agar file html kita tidak terlalu berantakan jika ingin menggunakan css.
18) a= untuk membuat link ke file lain, biasa disertai href untuk membuat link. (a href="")
19) img= untuk mengisi gambar ke html, biasa disertai dengan src untuk menampilkan gambarnya.
20) br= mirip enter :)
21) blockquote= ini untuk membuat enter tapi lebih jauh
22) form= untuk memulai membuat form.
23) input= untuk menjadi wadah pengisian form, biasa disertai type untuk jenis input yang ingin kita beri.
24) button= biasa button dipakai untuk submit
25) u= untuk underline text.
26) b= untuk menebalkan tulisan
27) textarea= mirip dengan input hanya saja tampilan pengisian lebih besar
28) table= untuk membuat table
29) tr= untuk membuat row baru di tabel
30) td= untuk membuat kolom baru di tabel

Masih ada banyak lagi tapi untuk sementara ini terlebih dahulu. :)
